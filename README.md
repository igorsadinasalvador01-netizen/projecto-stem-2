# 🎓 Projeto STEM — A HI FUNDHENI VAKITHI

Site institucional premium de captação de alunos para o **Projeto STEM**, uma instituição dedicada ao desenvolvimento académico e formação profissional, com foco em resultados reais.

> _"A HI FUNDHENI VAKITHI" — Vamos aprender juntos._

---

## 🎯 Objetivo do Projeto

Site profissional, moderno e altamente persuasivo desenhado como **máquina de conversão** para:

- Gerar contactos via **WhatsApp** e **chamadas telefónicas**
- Apresentar os serviços educacionais com clareza
- Transmitir **confiança, qualidade e autoridade**
- Converter visitantes em alunos

## ✅ Funcionalidades Implementadas

### 📑 Estrutura completa (12 secções)
1. **Top bar** com contactos e redes sociais
2. **Header sticky** com logo SVG + menu responsivo
3. **Hero** com headline persuasiva, CTAs, estatísticas animadas e cards flutuantes com o logo do Projeto STEM
4. **Barra de urgência** com countdown de 7 dias (auto-renovável via `localStorage`)
5. **Sobre** a instituição com badge de +10 anos
6. **Serviços** — 7 cards (1ª-6ª, 7ª-12ª destaque, Inglês, IA, Informática, Robótica, Alfabetização) + card CTA
7. **Preços** — 6 planos (todos 1.000 MT) com destaque no plano 7ª-12ª
8. **Diferenciais** — 6 motivos sobre fundo azul escuro
9. **Depoimentos** — 6 testemunhos com 5 estrelas
10. **CTA Final** — gradiente verde com 2 botões grandes + trust badges
11. **Contactos** — 4 cards + formulário inteligente
12. **Footer** completo em 4 colunas

### 🧲 Máquina de captação
| Elemento | Função |
|----------|--------|
| 🟢 Botão WhatsApp flutuante | Com pulso + tooltip + shake periódico (8s) |
| 📞 Botão chamada flutuante | Ligação directa para 826 311 111 |
| ⏰ Countdown 7 dias | Contagem regressiva auto-renovável |
| 🔥 Badges de urgência | "Vagas limitadas" + "Promoção por tempo limitado" |
| 📝 Formulário → WhatsApp | Envia dados pré-preenchidos para o WhatsApp |
| 🎯 CTAs personalizados | Cada serviço/plano tem mensagem específica |
| 👀 Exit intent | Agita o botão WhatsApp quando o utilizador vai sair |
| 📊 Contadores animados | 500+ alunos, 98% aprovação, 15+ professores |

### 📱 Responsividade
- Desktop, tablet e mobile
- Breakpoints: 1024px, 768px, 480px
- Menu hambúrguer no mobile
- Botões flutuantes adaptados para ecrãs pequenos
